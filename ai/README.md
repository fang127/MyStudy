# AI 应用开发学习

记录 AI 应用开发相关学习内容。当前目录主要作为后续学习入口，后续会按主题拆分笔记和实践项目。

## 学习内容

- 大语言模型（LLM）基础与 API 使用。
- Prompt 设计、结构化输出和工具调用。
- LangChain、LlamaIndex 等 AI 应用开发框架。
- RAG（检索增强生成）应用开发。
- AI Agent 设计与实现。
- 向量数据库使用，例如 Chroma、Milvus、Pinecone 等。
- AI 应用部署、评估、监控与优化。

## 规划目录

| 方向 | 说明 |
| --- | --- |
| LLM API | 模型调用、流式输出、错误处理、重试和限流。 |
| RAG | 文档解析、切分、嵌入、召回、重排和答案生成。 |
| Agent | 工具调用、任务规划、记忆和多轮状态管理。 |
| Evaluation | 数据集、自动评测、人工评测和回归测试。 |
