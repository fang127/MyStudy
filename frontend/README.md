# 前端开发学习

记录 HTML、CSS、JavaScript 基础语法和浏览器交互相关练习。

## 学习内容

- HTML 常见文本标签、属性、区块、表单。
- CSS 引入方式、选择器、常用属性、盒子模型、浮动和定位。
- JavaScript 基础语法、变量、数组、对象、函数、闭包、生成器。
- 浏览器对象、事件处理、DOM 操作和表格增删改查。
- Promise、async/await 等异步编程基础。

## 使用方式

- `.html` 文件可以直接用浏览器打开查看效果。
- `.js` 文件既可以在浏览器控制台中观察，也可以按需要使用 Node.js 运行。
- 公共样式集中放在 [CSS/styles.css](CSS/styles.css)。
