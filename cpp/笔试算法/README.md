# 笔试算法

这个目录用于整理笔试和在线编程题中的算法练习。目前目录还未加入具体题目，后续可以按题型或平台继续补充。

## 计划内容

| 分类 | 说明 |
| --- | --- |
| 数组与字符串 | 双指针、滑动窗口、前缀和、差分。 |
| 链表 | 反转链表、快慢指针、环检测、链表合并。 |
| 栈与队列 | 单调栈、单调队列、括号匹配。 |
| 哈希表 | 频次统计、去重、映射关系。 |
| 二叉树 | 遍历、构建、最近公共祖先、路径问题。 |
| 动态规划 | 背包、子序列、区间 DP。 |
| 图论 | BFS、DFS、拓扑排序、最短路。 |

## 建议组织方式

```text
笔试算法/
├── array/
├── linked_list/
├── tree/
├── dp/
└── graph/
```

每道题建议包含：

- 题目来源或题目描述。
- 解题思路。
- 复杂度分析。
- 可运行代码。
- 关键边界用例。
