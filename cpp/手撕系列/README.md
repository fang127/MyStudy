## 编译提示

单文件练习可以按需编译，例如：

```bash
g++ -std=c++17 -pthread threadPool.cc -o threadPool.out
./threadPool.out
```

不同文件可能需要不同的 C++ 标准或线程参数，建议先查看源码中的依赖和使用方式。

## 学习建议

1. 先写出最小可运行版本，再补边界条件。
2. 对内存函数重点验证空指针、长度为 0、重叠区间等情况。
3. 对并发题重点关注生命周期、异常安全、锁顺序和退出条件。
4. 智能指针练习要重点理解拷贝、移动、析构和自赋值。
