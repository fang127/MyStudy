# C++ 学习

记录 C++ 基础能力、并发编程、网络通信和高性能组件方向的学习代码。

## 学习内容

- C++ 基础语法、对象模型、内存管理与智能指针。
- STL 容器、算法、迭代器与常见数据结构。
- 现代 C++（C++11/14/17/20）特性。
- 多线程、线程池、锁、无锁结构与并发安全。
- 系统组件实践：时间轮、连接池、消息队列等。
- 网络通信与 RPC：gRPC、Protobuf、TLS/mTLS。
- 面试和笔试常见手撕题、排序、二叉树、单例等。

## 构建提示

- CMake 项目通常使用 `cmake -S . -B build` 和 `cmake --build build -j`。
- 单文件练习可以直接使用 `g++` 或按目录内说明运行。
- 依赖第三方库的项目需要先安装对应开发包，例如 gRPC、Protobuf、Folly、OpenSSL 等。
